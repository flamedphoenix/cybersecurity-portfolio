# A2 - Security Concepts in Public Space

## Digital Security

| # | Concept | Description |
|---|---------|-------------|
| 1 | Public WiFi | Free networks in cafes/malls with login portals to limit access |
| 2 | ATM Security | PIN pads with shields, card skimmer protections |
| 3 | Tap-to-Pay | Contactless payments use tokenisation to protect card details |
| 4 | Electronic Door Locks | Shops/banks use electronic entry with access control |

## Reflection
Public spaces use a mix of visible deterrents (guards, cameras, barriers) and subtle digital protections (encrypted payments, access-controlled WiFi) to keep people and assets safe.
