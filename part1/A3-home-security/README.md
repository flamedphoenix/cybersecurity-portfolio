#A3 - Security Concepts at Home

## Digital Security

| # | Concept | Description |
|---|---------|-------------|
| 1 | WiFi Password Protection | Home router uses WPA2/WPA3 encryption |
| 2 | Router Firewall | Built-in firewall blocks unauthorised inbound connections |
| 3 | Device PIN/Password | Phones and computers require authentication to unlock |
| 4 | Two-Factor Authentication | Used on accounts like email, banking, social media |
